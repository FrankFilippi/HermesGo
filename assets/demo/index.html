# ⬡ HermesGo

The Hermes Agent as a one-click Windows desktop app.

## Features
- Native desktop window (WebView2)
- Built-in PowerShell terminal (ConPTY)
- Workspace file drawer
- Bundled skills + Skill Market shortcut
- Logs under %LOCALAPPDATA%\hermes\logs
